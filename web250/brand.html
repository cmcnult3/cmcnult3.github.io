Concept

AdventurePal is an application built to help plan and track your recreational activities. Initial functionality of the application will include:

  • Creating custom lists of locations (i.e., cities, public lands, rivers, beaches, trails, etc..) the user intends to visit
  • Spaces for open ended comments about the purpose of the visit (i.e., specific experiences he/she want to have)
  • A journaling function to update during or after a visit is complete
  • A visit summary page to add visit dates, ratings, comments, images, etc.
  • A querying function to review locations based on visited/ want to visit status

Mission

AdventurePal aims to provide a service which allows individuals to plan and track recreational activities in order to maximize experiences wherever life takes them.
